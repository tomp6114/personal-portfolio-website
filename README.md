## 🚀 Overview

This repository contains the source code for my personal portfolio. The site was designed with a focus on modern web aesthetics, utilizing a clean **Professional Light Mode** theme with rich bronze accents and a customized typography system.

### Key Features
- **Responsive Design**: Fluid typography and flexible grid layouts that adapt seamlessly from mobile devices to ultrawide monitors.
- **Interactive Canvas Engine**: A lightweight, custom-built particle mesh network runs in the background of the hero section using Vanilla JavaScript `<canvas>`.
- **Modern CSS Styling**: Built entirely with Vanilla CSS using extensive custom properties (`:root` variables) for scalable theme management.
- **Performance Optimized**: Zero external framework dependencies. Minimal asset loading ensures incredibly fast parse and paint times.

## 🛠️ Technology Stack

- **HTML5**: Semantic document structure ensuring accessibility and SEO best practices.
- **CSS3**: Advanced CSS features including Grid, Flexbox, transitions, keyframe animations, and layered backgrounds.
- **JavaScript (ES6+)**: Custom particle animation engine, scroll-reveal observers, and mobile navigation logic.
- **Fonts**: [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) (Display), [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) (Code/Labels), and [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) (Hero/Headings).

## 💻 Running Locally

Since this is a static website with no build step required, you can simply clone the repository and open `index.html` in your browser.

```bash
# Clone the repository
git clone https://github.com/tomp6114/personal-portfolio-website.git

# Navigate into the directory
cd personal-portfolio-website

# Optional: Serve locally using a simple HTTP server (e.g., Python or Node)
npx serve .
# or
python3 -m http.server
```
